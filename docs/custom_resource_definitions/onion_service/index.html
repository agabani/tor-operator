Onion Service

An Onion Service is an abstraction of a Tor Onion Service.

A Tor Onion Service is a service that can only be accessed over Tor. Running a Tor Onion Service gives your users all teh security of HTTPS with the added privacy benefits of Tor.

Examples

Basic

The Tor Operator will create an Onion Service using an auto generated Onion Key.

#onionkey.yaml
apiVersion: tor.agabani.co.uk/v1
kind: OnionKey
metadata:
  name: onion-service-{{ include "example.fullname" . }}
  labels:
    {{- include "example.labels" . | nindent 4 }}
spec:
  auto_generate: true
  secret:
    name: onion-service-{{ include "example.fullname" . }}


#onionservice.yaml
apiVersion: tor.agabani.co.uk/v1
kind: OnionService
metadata:
  name: onion-service-{{ include "example.fullname" . }}
  labels:
    {{- include "example.labels" . | nindent 4 }}
spec:
  onion_key:
    name: onion-service-{{ include "example.fullname" . }}
  ports:
    - target: example:80
      virtport: 80

Onion Balance

The Tor Operator will create an Onion Service registered with an Onion Balance using an auto generated Onion Key.

#onionkey.yaml
apiVersion: tor.agabani.co.uk/v1
kind: OnionKey
metadata:
  name: onion-service-{{ include "example.fullname" . }}-onion-balance
  labels:
    {{- include "example.labels" . | nindent 4 }}
spec:
  auto_generate: true
  secret:
    name: onion-service-{{ include "example.fullname" . }}-onion-balance


#onionservice.yaml
apiVersion: tor.agabani.co.uk/v1
kind: OnionService
metadata:
  name: onion-service-{{ include "example.fullname" . }}-onion-balance
  labels:
    {{- include "example.labels" . | nindent 4 }}
spec:
  onion_balance:
    onion_key:
      hostname: 3evpjb3a5qxaubnzodlxlgn3ccp3achwwwtirgpznnl752oyg2oxekid.onion
  onion_key:
    name: onion-service-{{ include "example.fullname" . }}-onion-balance
  ports:
    - target: example:80
      virtport: 80
